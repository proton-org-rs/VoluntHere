# VoluntHere

A platform for engaged citizens, community leaders, and companies to support positive change within their local communities.

## Description

VoluntHere is a platform designed for engaged citizens, community leaders, and companies that want to support positive change within their local communities. It enables users to discover, organize, and participate in volunteer initiatives that directly benefit their neighborhoods.

## Features

- Discover local volunteer opportunities
- Organize community initiatives
- Connect with like-minded volunteers
- Track volunteer impact
- Support positive community change

## Getting Started

### Prerequisites

- Flask
- Auth0
- Solana

### Installation

1. Clone the repository
```bash
git clone https://github.com/proton-org-rs/VoluntHere.git
```

2. Navigate to the project directory
```bash
cd VoluntHere
```

3. Install dependencies
```bash
pip install -r requirements.txt
```

4. Create the database
```bash
python db_create.py
```

5. Run the app
```bash
python run.py
```
